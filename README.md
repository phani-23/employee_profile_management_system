# 👨‍💼 Employee Profile Management System

A modern **Full-Stack Employee Management Application** built using **React.js** and **Spring Boot**. This application enables users to efficiently manage employee profiles by performing Create, Read, Update, and Delete (CRUD) operations through an intuitive and responsive interface.

---

## ✨ Features

* ➕ Add new employee profiles
* 📋 View all employee records
* ✏️ Update employee information
* 🗑️ Delete employee profiles
* 🔍 Search employee details *(optional if implemented)*
* 📱 Responsive and user-friendly interface
* 🔗 REST API integration using Spring Boot

## 🚀 Getting Started

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/phani-23/employee_profile_management_system.git
```

### 2️⃣ Frontend Setup

```bash
cd frontend
npm install
npm start
```

🌐 React App:

```
http://localhost:3000
```

### 3️⃣ Backend Setup

```bash
cd backend
mvn spring-boot:run
```

🌐 Spring Boot API:

```
http://localhost:8080
```
